# AI Usage Report

## Tools Used & Use Cases
### ChatGPT
- Helped plan the Assignment 3 feature list based on the rubric.
- Suggested improvements for JavaScript structure, reusable functions, and state management.
- Helped generate and refine documentation files such as the README and technical documentation.
- Assisted with debugging ideas for form validation, filtering logic, and API error handling.

## Benefits & Challenges
### Benefits
- Provided a faster starting point for organizing the project and breaking the assignment into smaller tasks.
- Helped identify missing rubric requirements such as API integration, state management, and performance improvements.
- Suggested cleaner code structure and reusable functions.
- Made documentation writing faster and more organized.

### Challenges
- Some AI suggestions needed editing to better match the assignment requirements and the student’s own project style.
- AI-generated code still required testing in the browser to ensure the logic worked correctly.
- The sample API username needed to be changed manually if the student wanted to display personal repositories.

## Learning Outcomes
- Learned how to connect a portfolio website to an external API using `fetch`.
- Improved understanding of JavaScript array methods such as `filter()`, `sort()`, and `map()`.
- Practiced using `localStorage` to save theme and visitor name preferences.
- Improved knowledge of multi-step form validation and user-friendly status messages.
- Learned simple performance improvements such as lazy loading images and deferring JavaScript.

## Responsible Use & Modifications
- All AI-generated suggestions were reviewed before use.
- The code and text were edited to fit the assignment requirements and the student’s own portfolio content.
- The final structure, wording, and testing decisions were verified manually.
- Any unclear or unnecessary AI suggestions were removed or rewritten.
- The student remained responsible for understanding the code, checking correctness, and maintaining academic integrity.
