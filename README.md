# 202249820-OMAR ALYAMANI-assignment3

## Project Description
This project is an advanced personal portfolio web application created for Assignment 3. It builds on the earlier portfolio assignments by adding external API integration, more advanced JavaScript logic, state management, performance improvements, and updated documentation.

## Features
- Responsive portfolio layout using HTML and CSS
- Light/Dark mode toggle saved with `localStorage`
- Visitor name saving using `localStorage`
- Greeting based on the current time of day
- Visit timer that counts how long a user stays on the site
- Project list with:
  - category filtering
  - sorting by newest, oldest, or title
  - beginner/advanced level selection
- GitHub API integration to display live public repositories
- Contact form with advanced validation rules
- Smooth scrolling and mobile navigation toggle
- Performance-focused improvements such as small SVG assets, lazy-loaded images, and deferred JavaScript loading

## Setup Instructions
1. Download or clone the repository.
2. Open the project folder.
3. Run the project locally by opening `index.html` in a browser.
4. For best results, use a live server extension in VS Code so API requests and file loading behave consistently.
